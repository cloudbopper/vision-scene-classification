Classification Pipeline

We implemented the following steps in order to perform scene classification:
  • Explore the given Spatial Pyramid code and generate SIFT features for all the images
  • Create a random permutation of the file names in each class and pick 100 images in each class for training and use the rest of the images for testing
  • Building dictionary using:
    1. Codebook generated by K-means clustering
    2. Optimized codebook by using incremental codebook optimization on the K-means codebook(Algorithm 4.1)
  • Input to SVM classification:
    1. Implemented baseline method by exploring SpatialPyramid code: BuildHistograms for building histograms of all images after texton labeling(each SIFT feature assigned to exactly one cluster) and CompilePyramid for generating the pyramid
    2. Implemented LLC pipeline by modifying SpatialPyramid code:
  • SVM for training and testing, we used two modules:
    1. liblinear-1.96
    2. libsvm-3.20
  • Training SVM based on the two different pipelines:
    1. Baseline: Histogram intersection kernel is used for training SVM
    2. LLC: linear kernel is used for training SVM
  • Classify the test images using the two SVM methods
  • Classify the test images using the two SVM methods
